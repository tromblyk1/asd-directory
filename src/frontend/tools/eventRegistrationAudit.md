# Event Registration Link Audit Checklist

## Instructions:
For each event that says "Registration Required" or similar, manually visit the link and verify:

### ✅ GOOD LINK - Has at least one of these:
- [ ] Registration button/form visible on page
- [ ] "RSVP" button that works
- [ ] Contact email/phone prominently displayed for registration
- [ ] Clear instructions on how to register

### ❌ BAD LINK - Flag if:
- [ ] Event page exists but NO way to register
- [ ] Link is 404 / broken / dead
- [ ] Registration "coming soon" (note date)
- [ ] Page requires login to see registration
